About project

Test Guru is a project designed to create and manage quizzes. It provides a user-friendly interface for both quiz creators and quiz takers. With Test Guru, you can easily create, edit, and organize quizzes, as well as track participants' progress and generate insightful reports

Technical Stack

Test Guru is built using a modern and robust technical stack, including:

  • Programming Language: Ruby
  • Framework: Ruby on Rails
  • Database: SQLite
  • Front-end: HTML, CSS, JavaScript
  • User Interface Framework: Bootstrap
  • Testing Framework: RSpec
  • Version Control: Git
  • Task from: Thinknetica
